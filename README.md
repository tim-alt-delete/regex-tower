# About
Regex Tower is a simple webapp for practicing regular expressions. Test your regex skills in progressively challenging levels.

https://regex.timothypulliam.com

- Written in Python/Flask and Javascript
- CSS with Bulma
- Containerized runtime (Docker)
- Github Actions CI to auto build Docker image and push to Docker Hub.
- Progress is tracked via JWT
